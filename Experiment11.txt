Experiment 11: 
Write a shell script to check whether a given number is even or odd

file2.sh
#!/bin/bash
read -p "Enter a number: " num

if (( num % 2 == 0 ))
then
    echo "$num is Even"
else
    echo "$num is Odd"
fi
